ng2-optimizely-sample is a sample project for testing Optimizely experiments within a Angular 2 SPA.

Features

  • Angular 2 implementation based on angular2-seed.
  • Uses ngrx/store and ngrx/effects for application state management.
  • Mock "authentication" behavior to show/hide a call to action button on the homepage.
  • Exposes this state using ngrx/effects for use by Optimizely (conditonal activation mode).
